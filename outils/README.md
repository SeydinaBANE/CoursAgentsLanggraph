# Multi-Agent LangChain

Un projet pour créer des agents intelligents avec LangChain.

## Installation

```bash
uv venv --python 3.11
source .venv/bin/activate
uv pip install -r requirements.txt
```

## Utilisation

```bash
python outils/outils.py
```

## Outils

- **scrape_website()** - Scrape le contenu d'URLs
- **create_outline()** - Crée un plan structuré
- **read_document()** - Lit un document
- **write_document()** - Écrit un document
- **edit_document()** - Édite un document

## Prérequis

- Python 3.11+
- pip ou uv
